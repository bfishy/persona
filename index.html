Persona & Storyboarding Design

Constructing personas and user experience storyboards for public-view interfaces

Interface Overview

1. Entrace

The metro ticketing machines are commonly located near the entrace.

2. Start

Start page of the ticketing kiosk's interactive screen

3. Select Price

Select the ticket price that corresponds to your destination

4. Fare map

Users consult the fare map located above the ticketing machine if they are unsure about the fare for getting to their destination.

On the fare map, each station has a corresponding fare value.

5. Calculate Price

The machine calculates the price that the user needs to pay based on the individual ticket price and the number of tickets intended to be purchased.

6. Pay

User pays the charged amount by inserting coins or bills.
